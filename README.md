# Max Elytra Fly Speed

A Fabric mod to customize the maximum Elytra flight speed using a native Minecraft GameRule.

Part of the **Instant Gratification Collection** — mods that respect the player's time.

## Configuration

The mod registers a sovereign `GameRule`:

- `/gamerule max-elytra-fly-speed:max_elytra_fly_speed <Integer>`: Sets the maximum flight speed of Elytras in Blocks/Second. Default: `50` blocks/sec.
